# Course 1 — Preflop — D2: 3-Bet Spot Quiz (IP/OOP Sizing + Hand Classes)

**Purpose:** Train *what hands* 3-bet for value vs as bluffs, and *how big* to size in-position (IP) vs out-of-position (OOP).

## Assumptions
- 6-max NLHE cash, 100bb effective, no ante.
- Villain opens to **2.5bb** (unless stated).
- If you 3-bet:
  - **IP default:** ~**3.0–3.5×** the open.
  - **OOP default:** ~**4.0–5.0×** the open.

External sizing references (used as *general* heuristics, not holy law):
- RakeRace notes IP 3-bets are “usually… between **3–3.5×**” the open (solver-oriented discussion) and emphasizes adapting to open size. https://rakerace.com/news/poker-strategy/2024/07/17/choosing-our-bet-sizes-preflop-3-betting-and-4-betting
- BlackRain79 summarizes a common heuristic: 3-bet **~3× in position** and **~4× out of position** (preflop sizing cheat sheet). https://www.blackrain79.com/2022/08/preflop-bet-sizing.html

## Hand-class cheat sheet (for the quiz)
Use these labels in explanations/feedback:
- **Value 3-bet:** hands happy to get action (QQ+, AK as the core; add JJ/AQs vs looser opens).
- **Linear 3-bet:** strong-but-not-nuts broadways/pairs that do well vs continuing range (e.g., AQs, AJs, KQs, TT–JJ).
- **Polar bluff 3-bet:** hands that aren’t great calls but have blockers/playability (e.g., A5s–A2s, KTs/K9s sometimes).
- **Call:** hands that realize well IP but don’t love 4-bet wars (suited connectors, suited broadways, small pairs IP).
- **Fold:** dominated offsuit trash, weak gappers OOP, etc.

## Scoring rubric (Green / Yellow / Red)
Score each question on **two axes**: (1) Action, (2) Sizing (if 3-bet).

### Action score
- **Green (1.0):** matches answer key action.
- **Yellow (0.5):** close deviation (e.g., CALL vs MIX/3-bet with a hand that can plausibly do both).
- **Red (0.0):** clear leak (e.g., flatting premiums OOP vs aggressive players; 3-betting dominated offsuit hands).

### Sizing score (only if 3-bet)
- **Green (1.0):** within recommended band.
- **Yellow (0.5):** off by ~0.5–1.0× open but still functional.
- **Red (0.0):** wildly off (too small OOP giving great price; too big IP torching bluffs).

**Total per question:** average(Action, Sizing) when sizing applies; otherwise Action score.

## Answer key (baseline) + sizing
> Sizes given in **bb**.

### Versus CO open (Hero BTN = IP)
1) AKo — **3-bet (value)** to **7.5–8.5bb** (≈3–3.5×)
2) A5s — **3-bet (polar bluff)** to **7.5–8.5bb**
3) KQo — **CALL** (or **MIX** 3-bet vs wide CO); baseline **CALL**
4) JTs — **CALL** (plays great IP)
5) 77 — **CALL** (set-mine + IP playability)

### Versus BTN open (Hero SB = OOP)
6) QQ — **3-bet (value)** to **10–12.5bb** (≈4–5×)
7) AJs — **3-bet (linear/value-ish)** to **10–12.5bb**
8) KTs — **MIX** (often 3-bet as bluff) to **10–12.5bb**
9) T9s — **CALL** (some pools prefer more 3-bet; baseline call)
10) A9o — **FOLD** (dominated, ugly OOP)

### Versus HJ open (Hero CO = IP)
11) AQs — **3-bet (linear/value)** to **7.5–8.5bb**
12) KJs — **CALL** (or MIX 3-bet); baseline **CALL**
13) 66 — **CALL**

### Versus UTG open (Hero HJ = IP but vs tight range)
14) AKs — **3-bet (value)** to **7.5–8.5bb**
15) AQo — **FOLD** (or very tight mix call/3-bet depending on rake/pool; baseline **FOLD**)
16) TT — **CALL** (some mixes; baseline call)

### Open-size adjustments
17) CO opens **3.0bb**. Hero BTN A4s — **MIX** (3-bet less vs bigger open); if 3-bet: **9–10.5bb** (still ~3–3.5×)
18) CO opens **2.0bb**. Hero BTN KQs — **3-bet (linear)** to **6–7bb**

### Blind defense (often OOP)
19) BTN opens 2.5bb. Hero BB A5s — **MIX** (call or 3-bet bluff). If 3-bet: **10–12.5bb**
20) BTN opens 2.5bb. Hero BB KJo — **CALL** (defend vs wide opens; don’t torch it with a bluff 3-bet)

## Coaching feedback patterns

### Pattern 1 — Same sizing IP and OOP
**Signal:** Student uses 8bb from SB and 12bb from BTN.
**Coach:**
- “OOP you want bigger: deny equity, reduce IP advantage, and simplify SPR.”

### Pattern 2 — 3-betting too linear IP
**Signal:** 3-bets hands like JTs/77/KJs IP a lot.
**Coach:**
- “IP you can profitably call a ton. Save 3-bets for value + good blocker bluffs; keep your calling range alive.”

### Pattern 3 — Flatting premiums (esp. OOP)
**Signal:** CALL with QQ+/AK from SB/BB.
**Coach:**
- “Don’t give them a cheap flop with your strongest hands OOP. Build the pot and start the hand with initiative.”

### Pattern 4 — Bluffing dominated offsuit junk
**Signal:** 3-bets A9o/KTo/QJo OOP.
**Coach:**
- “Your bluffs should block strong continues and have postflop playability. Offsuit dominated hands do neither.”

### Pattern 5 — No open-size adjustment
**Signal:** Same 3-bet size regardless of open.
**Coach:**
- “Stop using a single number. Think in multiples: ~3–3.5× IP, ~4–5× OOP, then translate to bb.”
