# Course 1 — Preflop — D4: Final Assessment (Mix: RFI + 3-Bet + BB Defense)

**Purpose:** Verify the student can (1) RFI by position, (2) choose 3-bet action + size IP/OOP, and (3) defend BB vs steals.

## Assumptions
- 6-max NLHE cash, 100bb, no ante.
- Default open size: 2.5bb unless stated.
- Student answers in **≤ 6 seconds**.

## Scoring rubric (Green / Yellow / Red)
Each question yields 0–2 points.
- **Action (0–1):** Green=1, Yellow=0.5, Red=0
- **Sizing (0–1):** only if question includes a 3-bet size choice. Green=1, Yellow=0.5, Red=0

**Total points:** sum across questions.

**Grade mapping (suggested):**
- **Green Pass:** ≥ 85% of max points
- **Yellow Pass (needs review):** 70–84%
- **Red (repeat module):** < 70%

## Feedback macros (attach after grading)
- If **RFI errors** dominate → send D1 patterns (too loose EP / too tight LP).
- If **3-bet sizing** errors dominate → “multiples not numbers” reminder (~3–3.5× IP, ~4–5× OOP).
- If **BB overfolding** dominates → pot-odds reminder (small opens give BB a price).

## Assessment (25 questions)
### Section A — RFI (10)
1) Folds to you, **UTG**, hand **AQo**. Action? (OPEN/FOLD)
2) Folds to you, **UTG**, hand **99**.
3) Folds to you, **HJ**, hand **KQo**.
4) Folds to you, **HJ**, hand **ATs**.
5) Folds to you, **CO**, hand **JTo**.
6) Folds to you, **CO**, hand **A9o**.
7) Folds to you, **BTN**, hand **K7s**.
8) Folds to you, **BTN**, hand **T8s**.
9) Folds to you, **SB**, hand **QJs**.
10) Folds to you, **SB**, hand **AJo**.

### Section B — 3-bet decisions + sizing (10)
11) CO opens 2.5bb. Hero BTN **AKo**. (CALL/3-BET/FOLD) If 3-bet, size?
12) CO opens 2.5bb. Hero BTN **A5s**. If 3-bet, size?
13) CO opens 2.5bb. Hero BTN **JTs**.
14) BTN opens 2.5bb. Hero SB **QQ**. If 3-bet, size?
15) BTN opens 2.5bb. Hero SB **KTs**.
16) HJ opens 2.5bb. Hero CO **AQs**. If 3-bet, size?
17) UTG opens 2.5bb. Hero HJ **TT**.
18) CO opens **3.0bb**. Hero BTN **A4s**. If 3-bet, size?
19) CO opens **2.0bb**. Hero BTN **KQs**. If 3-bet, size?
20) BTN opens 2.5bb. Hero BB **A5s**. If 3-bet, size?

### Section C — BB defense (5)
21) BTN opens 2.5bb. Hero BB **QJo**. (3-BET/CALL/FOLD)
22) BTN opens 2.5bb. Hero BB **A9o**.
23) CO opens 2.5bb. Hero BB **KJo**.
24) BTN opens **3.0bb**. Hero BB **KTo**.
25) BTN opens **2.0bb**. Hero BB **J9s**.

## Answer key (baseline)
### Section A — RFI
1) UTG AQo — **FOLD** (baseline)
2) UTG 99 — **OPEN**
3) HJ KQo — **MIX** (score OPEN or FOLD as Yellow; baseline MIX)
4) HJ ATs — **OPEN**
5) CO JTo — **OPEN**
6) CO A9o — **MIX** (often open; Yellow if fold)
7) BTN K7s — **OPEN**
8) BTN T8s — **OPEN**
9) SB QJs — **OPEN**
10) SB AJo — **MIX**

### Section B — 3-bet + sizing
11) BTN vs CO, AKo — **3-bet**, size **7.5–8.5bb**
12) BTN vs CO, A5s — **3-bet (bluff)**, size **7.5–8.5bb**
13) BTN vs CO, JTs — **CALL**
14) SB vs BTN, QQ — **3-bet**, size **10–12.5bb**
15) SB vs BTN, KTs — **MIX** (3-bet bluff or call); if 3-bet: **10–12.5bb**
16) CO vs HJ, AQs — **3-bet**, size **7.5–8.5bb**
17) HJ vs UTG, TT — **CALL**
18) BTN vs CO (3.0bb open), A4s — **MIX**; if 3-bet: **9–10.5bb**
19) BTN vs CO (2.0bb open), KQs — **3-bet**, size **6–7bb**
20) BB vs BTN, A5s — **MIX**; if 3-bet: **10–12.5bb**

### Section C — BB defense
21) vs BTN 2.5bb, QJo — **CALL**
22) vs BTN 2.5bb, A9o — **FOLD**
23) vs CO 2.5bb, KJo — **FOLD**
24) vs BTN 3.0bb, KTo — **FOLD**
25) vs BTN 2.0bb, J9s — **CALL**

## Coaching feedback patterns (final)

### If they pass but sizing is sloppy
- “You’ve got the right actions, but your sizes are random. Think **multiples**: ~3–3.5× IP, ~4–5× OOP, then translate.”

### If they pass but RFI is too tight
- “You’re leaving money on the table late position. BTN/CO opens fund everything else.”

### If they fail due to BB overfolding
- “You’re paying full price for the BB then folding it away. Small opens give you pot odds — defend playable suited hands, ditch dominated offsuit trash.”

### If they fail due to bluff-spew
- “Bluffs need blockers + playability. If you can’t name the reason, it’s not in the range.”
